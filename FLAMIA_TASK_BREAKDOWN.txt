TASK 3.2: Implement JWT Authentication Filter

  Goal: Spring Security filter that extracts and validates JWT from requests.
  Required Files:
    security/JwtAuthenticationFilter.java
    security/UserPrincipal.java
  Dependencies: Tasks 3.1, 2.7 (UserRepository)
  Completion Criteria:
    - Extends OncePerRequestFilter
    - Extracts Bearer token from Authorization header
    - Skips validation for public endpoint paths
    - On valid JWT: loads User from DB, sets SecurityContext
    - On invalid JWT: clears SecurityContext, chain continues (returns 401)
    - On missing JWT: chain continues without auth (public access)
    - UserPrincipal implements Authentication with user ID, phone, role
